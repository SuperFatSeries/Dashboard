| :----------: | :--------------------------------------- |
|      范围      | web应用                                    |
|      级别      | 用户目标                                     |

|   涉众及其关注点    |学生课程基本需求           |
|     前置条件     | 学生正确访问课程主页                  |

|     发生频率     | 经常发生                                   |
